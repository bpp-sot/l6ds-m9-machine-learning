# Common Loss Functions

> A Loss Function (or Cost Function) is the mathematical equation an algorithm uses to explicitly calculate exactly how "wrong" its predictions are natively. 

## Regression Loss Functions

Algorithms like Linear Regression use these cleanly.

1. **Mean Squared Error (MSE):**
   * Heavily punishes large geometric outliers mathematically because the error is squared.
   * *Formula:* Average of $(True - Predicted)^2$
2. **Mean Absolute Error (MAE):**
   * Treats all errors exactly equally conceptually.
   * *Formula:* Average of $|True - Predicted|$

## Classification Loss Functions

Categorical algorithms like Logistic Regression use these natively.

1. **Log Loss (Cross-Entropy):**
   * Punishes an algorithm heavily if it mathematically predicts "99% confident" but is structurally wrong cleanly.
