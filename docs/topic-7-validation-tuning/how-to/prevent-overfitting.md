## Top Strategies
1.  **More Data:** The ultimate cure.
2.  **Regularisation:** Add penalties ($L1$, $L2$) in linear models.
3.  **Pruning:** Limit `max_depth` or `min_samples_leaf` in trees.
4.  **Early Stopping:** Stop boosting when validation error rises.
